# Setup Tools 101

## 1. Email

ฟังก์ชันนี้ให้คุณ ระบุที่อยู่อีเมล ที่ต้องการให้ระบบใช้สำหรับส่งข้อมูลสำคัญ
คุณสามารถ ลบ ที่อยู่อีเมลที่เคยเพิ่มไว้ได้ทุกเมื่อ หากไม่ต้องการรับการแจ้งเตือนไปยังอีเมลนั้นอีกต่อไป

:::info[ต้องการมีระบบธง/ยกมือ (Flagging) ใน Rudi ไหม?]
หากคุณต้องการให้ Rudi สามารถ จัดหมวดหมู่คำถาม หรือ ตอบคำถามพื้นฐาน ได้อย่างมีประสิทธิภาพมากขึ้น
คุณสามารถ ตั้งค่า ระบบธง/ยกมือ (Flagging) ได้ง่าย ๆ ผ่าน การตั้งค่า Email นี้ โดยใช้ Prompt พื้นฐานที่เราเตรียมไว้ให้ด้านล่างนี้
:::

## 2. RAG

ฟังก์ชันนี้ให้คุณ อัปโหลดเอกสาร (เช่น PDF, Word, หรือไฟล์ข้อความ) เข้าไปในฐานข้อมูลของระบบ เพื่อให้ระบบ RAG สามารถ ดึงข้อมูล จากเอกสารเหล่านั้นมาใช้ในการสร้างคำตอบ
ช่วยให้ AI สามารถตอบคำถามเฉพาะเจาะจงเกี่ยวกับข้อมูลภายในองค์กรหรือข้อมูลล่าสุดที่คุณเพิ่มเข้าไปได้ ทำให้คำตอบมีความถูกต้องและเป็นปัจจุบันมากขึ้น

:::info[ต้องการใช้ RAG กับไฟล์เอกสารของคุณไหม?]
หากคุณต้องการให้ Rudi สามารถ ดึงข้อมูลจากเอกสารต่าง ๆ ที่เกี่ยวข้องกับธุรกิจของคุณ
คุณสามารถ ตั้งค่า ระบบ RAG ได้ง่าย ๆ ผ่าน การตั้งค่า RAG นี้ โดยใช้ Prompt พื้นฐานที่เราเตรียมไว้ให้ด้านล่างนี้
:::

## 3. Fuzzy Search

เป็นกลไกการค้นหาที่สามารถ ค้นหาข้อมูลได้แม้ว่าคำที่สะกดจะไม่ตรงกันทั้งหมด (เช่น สะกดผิด, พิมพ์ตกหล่น, หรือมีตัวอักษรสลับตำแหน่งกัน)
ระบบจะคำนวณความใกล้เคียง (Similarity) ของคำค้นหากับข้อมูลในฐานข้อมูล และแสดงผลลัพธ์ที่ ใกล้เคียงที่สุด ออกมา
ช่วยให้ผู้ใช้ค้นหาไฟล์, ชื่อลูกค้า, หรือคำสั่งต่าง ๆ ได้ง่ายขึ้นมาก แม้จะพิมพ์ผิดไปบ้างก็ตาม

:::info[ต้องการใช้ Fuzzy Search กับข้อมูลของคุณไหม?]
หากคุณต้องการให้ Rudi สามารถ ค้นหาข้อมูลได้อย่างยืดหยุ่นมากขึ้น
คุณสามารถ ตั้งค่า ระบบ Fuzzy Search ได้ง่าย ๆ ผ่าน การตั้งค่า Fuzzy Search นี้ โดยใช้ Prompt พื้นฐานที่เราเตรียมไว้ให้ด้านล่างนี้
:::

## 4. Pandas

คุณสามารถ: 1. นำเข้า ข้อมูลตาราง (เช่น CSV) 2. จัดการ และ ประมวลผล ชุดข้อมูลขนาดใหญ่ได้ 3. สร้างรายงาน หรือทำการวิเคราะห์ข้อมูลเชิงลึกในรูปแบบตารางได้อย่างมีประสิทธิภาพ

:::info[ต้องการใช้ Pandas กับข้อมูลตารางของคุณไหม?]
หากคุณต้องการให้ Rudi สามารถ วิเคราะห์ข้อมูลตาราง หรือ ชุดข้อมูลขนาดใหญ่ ได้อย่างมีประสิทธิภาพมากขึ้น
คุณสามารถ ตั้งค่า ระบบ Pandas ได้ง่าย ๆ ผ่าน การตั้งค่า Pandas นี้ โดยใช้ Prompt พื้นฐานที่เราเตรียมไว้ให้ด้านล่างนี้
:::

# How to embed on WordPress

:::info
⬇️ ดาวน์โหลด Plugin Rudi V2 สำหรับการฝังโค้ดบน WordPress ⬇️

- [Rudi-Plugin V2.Zip](/downloads/rudi-plugin-V2.zip)
:::

### Plugin

1. เลือก Add New Plugin

![Add New Plugin](../static/img/WordPress/add_new_plugin.png)

2. เลือก Upload Plugin

![Upload Plugin](../static/img/WordPress/upload_plugin.png)

3. คลิกที่ Choose File และคลิกที่ Install Now

![Choose File](../static/img/WordPress/install_now.png)

4. เลือกไฟล์ rudi-plugin.zip

![Select File](../static/img/WordPress/select_file.png)

5. คลิกที่ Install Now

![Install Now](../static/img/WordPress/install_now.png)

6. คลิกเปิด Activate Plugin

![Activate Plugin](../static/img/WordPress/activate_plugin.png)

### Rudi Options

คลิกที่เมนู Rudi Options

![Rudi Options Menu](../static/img/WordPress/rudi_option.png)

- คัดลอก Embed Script แล้วคลิก Save Setting

![Embed Script](../static/img/WordPress/embed_code.png)

![Resut Embed](../static/img/WordPress/result.png)
